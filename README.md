# 🛍️ Vrinda Store Data Analysis

An interactive Excel-based data analytics project focused on analyzing sales performance, customer behavior, order trends, andbusiness insights for Vrinda Store.

<img width="677" height="267" alt="Dashboard Image" src="https://github.com/user-attachments/assets/10a4e69a-2868-489c-94c0-82d68d229675" />

## 📊 Project Overview

This project uses Microsoft Excel to analyze store sales data and create meaningful business insights through Pivot Tables, Charts, Dashboards, and Data Visualization techniques.

The goal is to identify sales trends, customer demographics, top-performing states, sales channels, and product categories to support data-driven decision making.

## ✨ Features

- Interactive Excel Dashboard
- Sales Performance Analysis
- Customer Demographics Analysis
- Order Status Analysis
- State-wise Sales Analysis
- Channel-wise Sales Analysis
- Category-wise Sales Analysis
- KPI-Based Reporting
- Pivot Tables & Pivot Charts

## 🛠️ Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Analysis
- Data Visualization

## 📈 Key Business Insights

- Identified top-performing states based on revenue.
- Analyzed customer purchasing patterns by gender and age group.
- Compared sales performance across multiple sales channels.
- Evaluated order status trends including delivered, cancelled, and refunded orders.
- Analyzed product category performance and revenue contribution.
